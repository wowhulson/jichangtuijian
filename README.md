## 📌 评估维度

当前推荐主要按这些标准来判断：

- **稳定性**：高峰期是否掉速、波动、失联
- **速度体验**：网页、视频、下载、多设备体验是否顺畅
- **线路质量**：三网表现、中转质量、晚高峰表现
- **平台兼容**：Windows / macOS / Android / iPhone 是否好用
- **流媒体支持**：是否适合视频、观影、常见海外内容访问
- **AI 工具访问**：是否适合 ChatGPT、Claude、Google 等工具
- **新手友好度**：订阅导入是否方便、上手是否简单
- **性价比**：套餐价格与长期使用成本是否合理

## 🧭 怎么选机场

如果你不知道怎么选，可以按需求来看：

### 想要稳定
优先看：
- 晚高峰表现
- 长期可用性
- 是否适合主用

### 想要性价比
优先看：
- 入门套餐价格
- 月付/季付是否合理
- 是否存在低价但明显不稳的问题

### 想看流媒体 / 4K 视频
优先看：
- 晚高峰速度
- 线路质量
- 是否容易出现卡顿或地区限制问题

### 想用 ChatGPT / Claude / Google 等 AI 工具
优先看：
- 海外访问稳定性
- 节点连通质量
- 是否容易遇到验证或访问异常

## 💡 新手建议

如果你是第一次接触机场，建议先记住这几条：

1. **先看稳定，再看价格**  
   太便宜的机场不一定真划算，稳定才是核心。

2. **不要被“节点很多”迷惑**  
   节点多不代表好用，关键还是实际可用性。

3. **优先选上手简单的**  
   对新手来说，订阅方便、客户端兼容，比参数更重要。

4. **从自己的场景出发**  
   你是日常浏览、观影、AI 工具，还是长期主用，需求完全不同。

## ❓ 常见问题 FAQ

### 机场是什么？
通常是提供代理节点、订阅和网络加速服务的一类产品。

### 怎么判断一个机场稳不稳？
看高峰期表现、长期可用性、线路质量，以及是否容易出现大面积掉速或失联。

### 协议重要吗？
重要，但对大多数人来说，最终还是看实际体验：稳不稳、快不快、好不好用。

### 新手最应该看什么？
优先看：
- 稳定性
- 易用性
- 平台兼容
- 订阅是否方便
